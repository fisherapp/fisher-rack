## Installation

Add this line to your application's Gemfile:

    gem 'fisher-rack'

And then execute:

    $ bundle

## Usage

### Rails 3.0 or later

To configure your application to track application stats run `rails g fisher:rack:install`. Run
`rails g fisher:rack:install -h` for more options.
